UI/UX DESIGN

The UI designer creates the visual and interactive experience that a user has with a product interface on an app or website. The UI designer is responsible for the screens, touchpoints & micro-interactions a User will interact with while using a virtual product.

UI/UX SKILLS
  • Creating User journey and User flow maps
  • Creating wireframes, prototyping and mockups
  • Using design software such Figma, Adobe XD, Adobe Photoshop, Adobe Illustrator, etc.

UI/UX DESIGN DESCRIPTION
  • Working together with the client and UX designer to establish a common vision
  • Researching your target audience and getting to know the user personas
  • Designing screens, buttons, scrollbars and other navigational elements
  • Prototyping and testing
  • Liaising with developers to ensure correct implementation
